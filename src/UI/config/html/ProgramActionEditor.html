Arguments

You can also use the following variables, which will be replaced with the actual values for the microphone:

  • ${microphone.name}
    • The microphone's name (example: Headset Microphone)
  • ${microphone.fullName}
    • The microphone's full name, including the controller's name (example: Headset Microphone (Razer Barracuda X))
  • ${microphone.state}
    • The current state of the microphone (Muted or Online)
  • ${microphone.isMuted}
    • Whether the microphone is muted or not (true or false)
  • ${microphone.hotkeyTriggered}
    • Whether the hotkey was triggered or not (true or false).
    if this is false, the microphone was un/muted by an external app
